# 🚀 Termux Learning Hub

**تطبيق تعليمي شامل لتعلم أوامر Termux - متعدد المنصات**

**Comprehensive educational application for learning Termux commands - Multi-platform**

---

## 📱 المشاريع المتضمنة | Included Projects

### 1. 📲 **Termux Learning Mobile App** (React Native + Expo)
تطبيق جوال متكامل لتعلم أوامر Termux مع دعم iOS و Android.

**Features:**
- ✅ 10 فئات من أوامر Termux
- ✅ 26+ أمر مع أمثلة وشروحات
- ✅ دعم كامل للعربية والإنجليزية
- ✅ نظام المفضلة والبحث
- ✅ تأثيرات هابتك
- ✅ Material Design UI

📁 **المسار:** `mobile-app/`

---

### 2. 🌐 **Termux Learning Web App** (React + Material-UI)
موقع ويب تفاعلي بتصميم Material Design مع دعم ثنائي اللغة.

**Features:**
- ✅ موقع ويب تفاعلي
- ✅ تصميم Material Design
- ✅ موضوع داكن بألوان Termux
- ✅ بحث متقدم
- ✅ نظام المفضلة
- ✅ PWA Ready

📁 **المسار:** `web-app/`

---

### 3. 📊 **Termux Database** (SQL + JSON)
قاعدة بيانات شاملة لجميع أوامر Termux بصيغة SQL و JSON.

**Contents:**
- ✅ 10 فئات رئيسية
- ✅ 26+ أمر Termux
- ✅ 20+ مثال عملي
- ✅ 8+ نصائح مفيدة
- ✅ 3+ حلول للأخطاء الشائعة

📁 **المسار:** `database/`

---

### 4. 🔧 **Sketchware Android App** (Java)
تطبيق Android كامل مكتوب بلغة Java جاهز للاستخدام في Sketchware.

**Features:**
- ✅ كود Java كامل
- ✅ SQLite Database
- ✅ Firebase Integration
- ✅ Material Design

📁 **المسار:** `android-app/`

## 🚀 البدء السريع | Quick Start

```bash

cd termux-learning-hub

cd web-app/termux-learning-web
npm install
npm start
```
